# ByeByeDPI iOS (черновик)

Этот репозиторий содержит каркас iOS-приложения для подбора команд ByeByeDPI под целевые домены.

## Что уже есть

- Главный экран с кнопкой включения/выключения VPN.
- Экран настроек:
  - запуск VPN при старте приложения;
  - переключатель IPv6;
  - переключатель DNS и поле адреса;
  - стратегия подбора команд;
  - экран подбора стратегий с запуском и прогрессом;
  - список пресетов команд;
  - список доменов YouTube/Google;
  - отображение версии.

## Что предстоит добавить

- Реальная интеграция с VPN-туннелем и применением команд в зависимости от выбранной стратегии.
- Хранение настроек и автозапуск.
